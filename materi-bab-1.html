Dasar HTML

HTML (HyperText Markup Language) merupakan sebuah bahasa markup, bukan bahasa pemrograman. Bahasa markup (Indonesia: markah) adalah bahasa yang mengombinasikan teks dan informasi tambahan mengenai teks tersebut. Dikatakan markup language karena HTML berfungsi untuk ‘memperindah’ file teks biasa untuk ditampilkan pada program web browser hal ini dilakukan dengan menambahkan elemen atau sering disebut sebagai tag-tag pada file teks biasa tersebut.
HTML diciptakan oleh Tim Berners-Le dengan menggunakan konsep hypertext. HTML sekarang diorganisasi oleh sebuah badan lembaga khusus bernama World Wide Web Consortium (W3C). Versi terbaru dari HTML saat ini adalah HTML 5. Beberapa fitur yang tersedia pada HTML5 antara lain sebagai berikut :

  • Canvas
    Memungkinkan pembuatan gambar dalam kanvas sehingga, gambar tidak lagi diambil dari gambar utuh, melainkan bisa disusun sendiri, seperti menggunakan program Paint
  • Header
    Berguna untuk menyatakan suatu judul, yang bisa diisi dengan elemen logo dan nama perusahaan
  • Footer
    Kebalikan dari header. Sebagai catatan kaki, elemen ini berguna untuk menaruh informasi di bagian bawah halaman web
  • Time
    Elemen ini berguna untuk menyajikan informasi tentang waktu
  • Audio
    Memungkinkan penyajian player untuk memutar suara
  • Video
    Memungkinkan pengguna untuk memainkan film

Struktur Dasar HTML

Script mendasar untuk membangun halaman web adalah HTML. HTML merupakan bahasa penanda (markup) pada dokumen teks. Penandaan (markup) dalam HTML menggunakan ‘ < ‘ tanda lebih kecil dan ‘ > ‘ tanda lebih besar, perintah dalam HTML disebut dengan tag. Tag atau perintah dalam HTML menggunakan perintah yang berpasangan, ditandai dengan <…> dan diakhiri </…>. Setiap dokumen HTML memiliki struktur dasar atau susunan file sebagai berikut:
Struktur file HTML diawali dengan sebuah tag <html> dan ditutup dengan tag </html>. Di dalam tag ini terdapat dua buah bagian besar, yaitu yang diapit oleh tag <head>...</head> dan tag <body>...</body>.

Head
Head merupakan bagian awal dari struktur HTML. Penggunaan Head bersifat optional dalam HTML, namun penggunaan Head yang benar dapat meningkatkan kinerja dokumen HTML. Selain berisi title atau judul dokumen pada. Bagian head dapat menetapkan judul halaman, menghubungkan file eksternal seperti CSS dan JavaScript, menambahkan meta tag, dan bagian head terdapat perintah lain yang tidak ditampilkan dihalaman browser
Elemen bagian head terdiri dari :

  • Title
    Elemen title digunakan untuk menentukan judul halaman web yang akan ditampilkan pada tab atau judul jendela browser. Contohnya :<title>Judul Halaman</title>
  • Base
    Elemen base digunakan untuk menentukan basis URL sebuah dokumen.Elemen ini membantu mengatur URL dasar yang digunakan sebagai acuan untuk menghubungkan URL relatif dalam dokumen HTML. Contohnya : <base href=“url”>
  • Link
    Elemen link digunakan untuk menghubungkan file eksternal ke halaman web, seperti file CSS atau file ikon. Contohnya : <link rel = "stylesheet" href="style.css">
  • Meta
    Elemen meta digunakan untuk menginformasikan browser terkait keyword untuk search engine, menentukan diskripsi halaman, penulis dokumen, terakhir diumbah dan meta data lainnya. Contohnya : <meta charset="UTF-8">
  • Script
    Elemen script digunakan untuk menyisipkan kode JavaScript ke halaman web. Biasanya, Anda juga dapat menentukan atribut src untuk menghubungkan file JavaScript eksternal. Contoh script : <script src="script.js" defer></script>

Body
Bagian body digunakan untuk menyimpan informasi atau data yang akan dipublish pada halaman browser atau web page. Properti document diatur melalui atribut-atribut yang terdapat dalam elemen. Sebagai contoh adalah pengaturan warna latar belakng halaman, wana teks, warna link dan lain-lain.

Penjelasan :
  • Sebuah dokumen HTML5 diawali dengan <!DOCTYPE HTML>
  • Pasangan <html>...</html> menyatakan awal dokumen HTML
  • Di dalam tag <html>...</html> terdapat pasangan tag <head>...</head> dan tag <body>...</body>
  • Pasangan tag <head>...</head> menyatakan bagian judul dokumen HTML. Isinya paling tidak berupa pasangan tag <title>...</title>
  • Pasangan tag <body>...</body> menyatakan bagian tubuh dokumen
  • Teks "Belajar HTML 5" adalah konten yang akan ditampilkan pada halaman web ini
